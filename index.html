About

Programming { }

I am a full-stack developer. I started my journey to learn computer programming a few years ago in attempt to create a video game. My ability to code was not always constant but I continue to learn despite the setbacks. So far I am able to use: Html, CSS, JavaScript, Node.Js, PHP, Dart, Flutter and C#. I make android apps, websites and computer programs. I plan to continue learning different languages so that I can be a better both web development and software development.

Animating ʕ•ᴥ•ʔ

I've been an animator for almost a decade. I am capable of creating both 2D and 3D animations for short films, presentations and video games. I am capable of using industry standard programs like Autodesk Maya, however I do most of my work using Blender for Modeling, Rigging, Animating, Editing and Compositing. Currently I'm learning to create realistic graphics and simulations.

Video Editing

I've been editing Videos for a few years now. I started with editing my own work then further branched out into editing for friends and Youtube Channels. During the Pandemic I started editing videos for my Church. Since then I've been the official editor. I use DaVinci Resolve, Olive and Blender.

Graphic Design

I started my graphic design journey in 2018. I have completed projects for schools and small businesses.
